# 🦇 Better Bats

**Better Bats** overhauls the vanilla Bat into a dynamic, ecosystem-driven entity. It turns the useless ambient creature into a swarming, light-sensitive, and symbiotic neighbor.

## ✨ Features

- 🌪️ **Hive Mind (Social Swarms)**: Bats organize into cohesive swarms using the DasikLibrary Hive Mind system.
- 💩 **Guano Fertility**: Resting bats passively fertilize farmland below, providing growth boosts to crops.
- 💡 **Phototaxis**: Bats are attracted to bright artificial light (>12 brightness), circling lanterns and torches to hunt insects.
- 🛡️ **Pest Control**: Bats aggressively dive-bomb and one-shot Silverfish and Endermites.
- 📢 **Acoustic Panic**: Loud noises (explosions, mining) wake resting swarms instantly.

## 📋 Requirements

- **Minecraft**: 26.1.2
- **Fabric Loader**: ≥ 0.19.1
- **Fabric API**: 0.145.4+26.1.2
- **DasikLibrary**: 1.6.9+build.22 (Standalone)

## 🛠️ Installation

1. Download the latest `.jar` from [Modrinth](https://modrinth.com/mod/better-bats) or [CurseForge](https://www.curseforge.com/minecraft/mc-mods/better-bats).
2. Place it in your `mods` folder alongside Fabric API and DasikLibrary.
